The SMTP implement based on LiteRT.

> **Currently only the server-side sending is implemented, which means no delegate works.**

## Installation

```sh
npm install @litert/smtp --save
```

## Requirements

- Node.js v8.x (Or newer)
- TypeScript v3.1.x (Or newer)
